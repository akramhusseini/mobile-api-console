Set up Browser capture

Three steps: this dialog enables Browser source, then you load the extension and click its icon on any page you want to capture. No URLs to type.

  1. Load the extension (once). Open brave://extensions (or chrome://extensions) → turn on Developer mode → Load unpacked → choose this folder:
    …/extension
  2. On your app, click the extension icon → “Capture this site.” That grants permission and starts capture. Pin the icon (🧩 menu → pin) so it's always handy.
  3. Watch it here. Pick Browser in the Source dropdown above — your app's API calls stream in with request & response bodies.
Advanced — launch a separate clean browser instead

Spins up an isolated browser profile with the extension preloaded. Most people don't need this: that profile has no login session, so apps requiring sign-in show a blank page. Prefer the two steps above.

  1. Browser source enabled in the console
  2. Browser launched with the unpacked extension
  3. In the extension tab, click Save, then Allow
  4. Select Browser and confirm one POST row with bodies
